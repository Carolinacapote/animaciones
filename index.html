Transformaciones

Las transformaciones son una de las características de CSS más interesantes la cual permite realizar efectos visuales 2D y 3D. Con ellas podemos hacer cosas como mover elementos, rotarlos, aumentarlos o disminuirlos y otras transformaciones relacionadas o combinadas. 

            
              .astronaut-img {
                border-radius: 2rem;
                width: 20rem;
                height: 20rem;
                margin: 0 auto;
                transform: rotate(20deg);
              }

Tipos de transformaciones

Timing functions:

La función de tiempo de animación especifica la curva de velocidad de una animación. La curva de velocidad define el TIEMPO que usa una animación para cambiar de un conjunto de estilos CSS a otro

Translate():

Translate permite mover un elemento en el eje X o Y, incluso en ambos, esto quiere decir que con CSS podríamos llevar un elemento desde un punto A a un punto B

            
              .translate-image:hover {
                transform: translate(0px, -100%);
                transition-duration: 2s;
              }
              .translate-image:not(:hover) {
                  transform: translate(0px, 0px);
                  transition-duration: 2s;
              }
Rotate, Scale, Skew y Matrix():

Rotate: Ayuda a girar un elemento.

Scale: Se pude agrandar o achicar un elemento.

Skew: Nos ayuda a deformar un elemento tanto en eje X como en Y.

Matrix: Esta propiedad es un poco más compleja, nos ayuda a rotar, escalar, deformar, esto se hace através de coordenadas.

            
              .scale-image {
                transition: transform 1s ease-in-out;
              }
              .scale-image:hover {
                  transform: scale(0.5);
              }
Transform origin:

Siempre utilizamos transform, el origen de esta transformación va ser en todo el centro, lo que indica que en caso que queremos rotar un elemento, rotará desde su punto de origen central, con css podemos cambiar ese punto de origen con transform origin.

            
              .transform-origin {
                margin: 0 auto;
                width: 30%;
                transition: transform 2s ease-in-out;
                transform-origin: left bottom;
              }          
              .transform-origin:hover {
                  transform: rotate(360deg);
              }
Transform de estilo y perspectiva:

En css también podemos hacer transformaciones en 3D, esto significa que podemos ajustar la perspectiva y el estilo de un componente. Este tipo de transformación lo que implica es que ya no tendremos un plano donde manejaremos un eje X y uno Y sino también un eje Z, que es la profundidad del elemento.

Transiciones

Son una forma de añadir efectos suaves a través de la propiedad 'transition', que permite controlar la duración y el tipo de transición entre los estados de una propiedad.

            
// transition: [property] [duration] 
               [timing-function] [delay];

transition-property.
transition-duration: Puede estar en segundos o milisegundos.
transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out | step-start | step-end | steps(int, start | end) | cubicbezier(n, n, n, n)
transition-delay: Tiempo de espera antes de que comience la transición.

Parallax

Es una técnica en la que el fondo se mueve a una velocidad distinta que la del contenido. El resultado es un ligero efecto de profundidad. Crea un efecto 3D.
Inventado por Disney para dotar a sus películas de dibujos animados de una ligera tridimensionalidad, intentando lograr con ello un mayor realismo. El invento se bautizó como «cámara multiplano»

Animaciones

Las animaciones CSS, al igual que las transiciones, crean movimientos y cambios graduales en los elementos de nuestra página.


La principal diferencia es, que una animación CSS tiene estados o frames (tantos como queramos) y por ello, son más complejas que las transiciones.

Propiedades de una animación

Propiedad ¿Para qué sirve?
animation-name Nombre con el cual identificamos la animación Obligatoria
animation-duration Tiempo que tardará en completarse toda la animación Obligatoria
animation-iteration-count Número de veces que se repite Opcional
animation-timing-function Define el progresoa lo largo de la duración de la animación. Define la curva de velocidad Opcional
animation-delay Tiempo de espera para que se ejecute la animación Opcional
animation-direction Especifica cómo debe reproducirse la animación (dirección normal, en reversa o en ciclos alternos) Opcional
animation-fill-mode Setea el estilo que tendrá el elemento animado cuando la animación no se está ejecutando Opcional
animation-play-state Pausa o inicia la animación Opcional
animation Forma corta de escribir todas las propiedades de la animación en una sola

@keyframes

Regla donde se define la animación, que propiedad(es) del elemento se van a cambiar y en que momento.

from - to

@keyframes change-color2 {
from { background: red; } /* Primer fotograma */
to { background: yellow; } /* Último fotograma */
}
/* Clase a animar */
.animated-2 {
width: 40vh;
height: 40vh;
border-radius: 50%;
animation: change-color2 3s linear infinite;
}
Porcentajes
