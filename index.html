Transformaciones

Las transformaciones son una de las características de CSS más interesantes la cual permite realizar efectos visuales 2D y 3D. Con ellas podemos hacer cosas como mover elementos, rotarlos, aumentarlos o disminuirlos y otras transformaciones relacionadas o combinadas. 

          
            .astronaut-img {
              border-radius: 2rem;
              width: 20rem;
              height: 20rem;
              margin: 0 auto;
              transform: rotate(20deg);
            }

Tipos de transformaciones

Timing functions:

La función de tiempo de animación especifica la curva de velocidad de una animación. La curva de velocidad define el TIEMPO que usa una animación para cambiar de un conjunto de estilos CSS a otro

Translate():

Translate permite mover un elemento en el eje X o Y, incluso en ambos, esto quiere decir que con CSS podríamos llevar un elemento desde un punto A a un punto B

          
            .translate-image:hover {
              transform: translate(0px, -100%);
              transition-duration: 2s;
            }
            .translate-image:not(:hover) {
                transform: translate(0px, 0px);
                transition-duration: 2s;
            }
Rotate, Scale, Skew y Matrix():

Rotate: Ayuda a girar un elemento.

Scale: Se pude agrandar o achicar un elemento.

Skew: Nos ayuda a deformar un elemento tanto en eje X como en Y.

Matrix: Esta propiedad es un poco más compleja, nos ayuda a rotar, escalar, deformar, esto se hace através de coordenadas.

          
            .scale-image {
              transition: transform 1s ease-in-out;
            }
            .scale-image:hover {
                transform: scale(0.5);
            }
Transform origin:

Siempre utilizamos transform, el origen de esta transformación va ser en todo el centro, lo que indica que en caso que queremos rotar un elemento, rotará desde su punto de origen central, con css podemos cambiar ese punto de origen con transform origin.

          
            .transform-origin {
              margin: 0 auto;
              width: 30%;
              transition: transform 2s ease-in-out;
              transform-origin: left bottom;
            }          
            .transform-origin:hover {
                transform: rotate(360deg);
            }
Transform de estilo y perspectiva:

En css también podemos hacer transformaciones en 3D, esto significa que podemos ajustar la perspectiva y el estilo de un componente. Este tipo de transformación lo que implica es que ya no tendremos un plano donde manejaremos un eje X y uno Y sino también un eje Z, que es la profundidad del elemento.
